# Employee Information Form

Official bilingual employee information form for collecting and reviewing employee data before migration to ZenHR.

## v4 redesign

- Formal employee-facing design with no technical marketing copy.
- Arabic/English interface switch while keeping the same validation rules.
- Full-screen management dashboard with KPIs, filters, tabs, import tools, and Excel exports.
- Hidden admin access by double-clicking the HR mark.
- Firebase Auth and Firestore workflow preserved.
- Excel exports preserved for employees, family members, and assets.

## Important deployment note

This is the GitHub-safe version. Do not upload private employee seed files, salary data, national IDs, social security information, or original Excel files to a public repository.
